## 🎮 HOW TO USE

### Create Email
1. Click template → Editor opens
2. Edit template blocks or delete them
3. Add new blocks from left panel
4. Customize each block in right panel
5. Drag to reorder
6. Save Draft to persist

### Edit Block
1. Click any block in canvas (center)
2. Settings appear in right panel
3. Modify content/colors/sizes
4. Changes instant & live

### Save Work
1. Click "Save Draft" button
2. See "✓ Draft Saved" confirmation
3. Browser localStorage saves email
4. Can close and resume later

## 🆘 TROUBLESHOOTING

**Q: Block settings not updating?**  
A: Click block again to refresh, or scroll in settings panel

**Q: Can't drag blocks to reorder?**  
A: Look for small drag handle on left side of block, drag from there

**Q: Draft not saving?**  
A: Wait for "✓ Draft Saved" toast, check browser storage

**Q: Colors look different on mobile?**  
A: Email width is fixed at 600px, responsive text used

**Q: Want to start over?**  
A: Click "← Back" then select template again
